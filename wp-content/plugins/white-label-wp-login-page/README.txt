== Description ==

**Install and activate the plugin and change the default style of WordPress login with a unique white-label style.**

[Try Out the Live Demo ‚ü∂](https://ozanwp.com/wp-login.php "live demo")

== Installation ==

1. Upload the entire white-label-wp-login-page folder to the /wp-content/plugins/ directory, or download and install automatically through your admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
